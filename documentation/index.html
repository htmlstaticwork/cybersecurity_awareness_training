1. Installation Guide

Follow these steps to set up the project environment.

Step 1: Local Environment

1

Extract the provided .zip file to your desired workspace.

2

Open the folder in **VS Code** or any preferred code editor.

3

Use the **Live Server** extension (VS Code) to launch the project locally at http://127.0.0.1:5500.

Step 2: Hosting Deployment

Since this is a static website, you can host it on any modern platform:

  • • Netlify: Drag and drop the root folder.
  • • Vercel: Push to GitHub and connect the repository.
  • • GitHub Pages: Enable via repository settings.

Folder Structure

/root

  └── ...             # Module specific HTML files

2. Customization Guide

Modify the look and feel of the platform by editing the CSS and Content.

Colors (CSS Variables)

Open css/style.css (Global styles) and locate the :root selector.

:root {
  --primary: #00d2ff;
  --secondary: #0072ff;
  --dark-bg: #0a0b10;
}

Typography

Fonts are imported from Google Fonts in the HTML head. To change fonts, update the link and font-family properties.

Default: Outfit (Headings) & Inter (Body).

Content Replacement

To update text or links, search for the following keywords in the HTML files:

  • • <!-- TODO: Replace client content --> - Marks areas for custom text.
  • • href="#" - Update navigation and button links.
  • • <img src="..."> - Replace images in the /assets or /images folder.

3. Page Structure

A breakdown of all the pages included in this documentation and their purpose.

Home (index.html)

The corporate landing page featuring the hero section, training overview, and value proposition.

Dashboard (dashboard.html)

The central hub for employees to track their progress, view active modules, and access reports.

Training (training.html)

Interactive module interface where the actual cybersecurity lessons and assessments take place.

Analytics (risk-analytics.html)

Admin panel to visualize company-wide security risk scores and completion rates.

Threats (threats.html)

A specialized library of current cyber threats (Phishing, Ransomware, etc.) for education.

Contact (contact.html)

Support channel for reporting issues or requesting advanced training assistance.
